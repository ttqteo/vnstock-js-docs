## So sánh tính năng giữa `vnstock3` và file test `vnstock-js`

| **Danh mục**              | **Tính năng**                         | **`vnstock3`**                                                      | **`vnstock-js`** |
|---------------------------|----------------------------------------|--------------------------------------------------------------------------------|-------------------------------|
| **Khởi tạo**              | Tạo đối tượng làm việc                 | `stock = Vnstock().stock(symbol='ACB', source='VCI')`                          | ❌                            |
| **Thông tin niêm yết**    | Danh sách công ty                      | `stock.listing.all_symbols()`                                                 | ✅ `listing.allSymbols()`     |
|                           | Danh sách các mã chỉ số                | Chưa bổ sung                                                                   | ❌                            |
|                           | Mã CP theo sàn                         | `stock.listing.symbols_by_exchange()`                                         | ✅ `listing.symbolsByExchange()` |
|                           | Mã CP theo ngành icb                   | `stock.listing.symbols_by_industries()`                                       | ✅ `listing.symbolsByIndustries()` |
|                           | Mã CP theo nhóm                        | `stock.listing.symbols_by_group('VN30')`                                      | ✅ `listing.symbolsByGroup()` |
|                           | Danh sách ngành icb                    | `stock.listing.industries_icb()`                                              | ✅ `listing.industriesIcb()`  |
| **Thông tin công ty**     | Tạo đối tượng                          | `company = Company(symbol='TCB')`                                             | ❌                            |
|                           | Tổng quan                              | `company.overview()`                                                          | ❌                            |
|                           | Hồ sơ                                  | `company.profile()`                                                           | ❌                            |
|                           | Cổ đông                                | `company.shareholders()`                                                     | ❌                            |
|                           | Giao dịch nội bộ                       | `company.insider_deals()`                                                    | ❌                            |
|                           | Công ty con                            | `company.subsidiaries()`                                                     | ❌                            |
|                           | Lãnh đạo                               | `company.officers()`                                                         | ❌                            |
|                           | Sự kiện                                | `company.events()`                                                           | ❌                            |
|                           | Tin tức                                | `company.news()`                                                             | ❌                            |
|                           | Cổ tức                                 | `company.dividends()`                                                        | ❌                            |
| **Báo cáo tài chính**     | Khởi tạo                               | `report = Finance(symbol='ACB', period='year')`                              | ❌                            |
|                           | Chỉ số tài chính                       | `report.ratio(lang='vi')`                                                    | ❌                            |
|                           | Bảng CĐKT                              | `report.balance_sheet(lang='vi')`                                            | ✅ `financials.balanceSheet()` |
|                           | Báo cáo kinh doanh                     | `report.income_statement(lang='vi')`                                         | ❌                            |
|                           | Lưu chuyển tiền tệ                     | `report.cash_flow(lang='vi')`                                                | ❌                            |
| **Giá**                   | Bảng giá                               | `stock.trading.price_board(['ACB'])`                                          | ✅ `trading.priceBoard()`     |
|                           | Giá lịch sử (điều chỉnh)               | `stock.quote.history(start='2020-01-01', end='2024-12-31')`                   | ✅ `quote.history()`          |
|                           | Lịch sử khớp lệnh (intraday)           | `stock.quote.intraday(symbol='ACB')`                                          | ❌                            |
|                           | Giá vs khối lượng khớp                 | `stock.quote.price_depth('ACB')`                                              | ❌                            |
| **Thị trường quốc tế**    | FX (nến)                                | `fx.quote.history(start='2024-02-28', end='2024-05-10', interval='1D')`       | ❌                            |
|                           | Crypto (nến)                           | `crypto.quote.history(start='2023-01-01', end='2024-12-31', interval='1D')`   | ❌                            |
|                           | Chỉ số (nến)                           | `index.quote.history(start='2023-01-01', end='2024-12-31', interval='1D')`    | ❌                            |
