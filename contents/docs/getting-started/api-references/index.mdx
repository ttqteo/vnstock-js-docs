# Hàm đang có và đang phát triển `vnstock-js`

## Khởi tạo đối tượng

Chúng ta có 2 cách khởi tạo đối tượng

1. Cơ bản

Hàm đã được tạo sẵn để sử dụng một số tính năng cơ bản

```ts
import { stock, commodity } from 'vnstock-js';
```

2. Nâng cao

Hàm này sẽ sử dụng đầy đủ tính năng từ vnstock-js, nhưng cách dùng sẽ phức tạp hơn so với cơ bản

```ts
import vnstock from 'vnstock-js';
```

## Hàng Hoá

Hiện tại đang có vàng, từ 3 nguồn: BTMC (Bảo Tín Minh Châu), giavang.net, và SJC

1. Cơ bản

```ts
import { commodity} from 'vnstock-js';

const goldBTMC = await commodity.gold.priceBTMC();
const goldGiaVangNet = await commodity.gold.priceGiaVangNet();
const goldSJC = await commodity.gold.priceSJC();
```

2. Nâng cao

```ts
import vnstock from 'vnstock-js';

const goldBTMC = await vnstock.commodity.goldPriceBTMC();
const goldGiaVangNet = await vnstock.commodity.goldPriceGiaVangNet();
const goldSJC = await vnstock.commodity.goldPriceSJC();
```
