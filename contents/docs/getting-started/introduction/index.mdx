Chào mừng bạn đến với **vnstock-js**!

Thư viện JavaScript/TypeScript lấy dữ liệu thị trường chứng khoán Việt Nam, tối ưu cho web.

## Tổng Quan

**vnstock-js** là gói npm mã nguồn mở, cung cấp dữ liệu chứng khoán VN với output đã chuẩn hóa (giá chia 1000, camelCase, ISO dates).

👉 [vnstock-js - npmjs.com](https://www.npmjs.com/package/vnstock-js)

### Tính Năng

| **Tính năng** | **Mô tả** |
|---------------|-----------|
| Báo Giá - Quote | Dữ liệu giá lịch sử OHLCV theo nhiều timeframe |
| Giao Dịch - Trading | Bảng giá realtime, top tăng/giảm |
| Niêm Yết - Listing | Danh sách mã, sàn, ngành ICB |
| Tài Chính - Financials | Bảng cân đối, KQKD, lưu chuyển tiền tệ |
| Công Ty - Company | Thông tin, cổ đông, ban lãnh đạo, sự kiện |
| Sàng Lọc - Screening | Lọc cổ phiếu theo tiêu chí tài chính |
| Chỉ Báo - Indicators | SMA, EMA, RSI |
| Hàng Hoá | Giá vàng (BTMC, SJC, GiaVang.net), tỷ giá VCB |
| Realtime | WebSocket dữ liệu giá realtime |
