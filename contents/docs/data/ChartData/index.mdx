## Các hàm sử dụng

Cơ bản:
1. `stock.quote()`
2. `stock.index()`

Nâng cao:
1. `vnstock.stock.quote.history()`

## Cấu trúc dữ liệu

Output là `QuoteHistory[]` — mảng các object:

```typescript
[
  { date: "2024-01-15", open: 25.5, high: 26.0, low: 25.0, close: 25.8, volume: 1000000 },
  { date: "2024-01-16", open: 25.8, high: 26.2, low: 25.3, close: 26.1, volume: 1200000 },
]
```

| Trường | Kiểu | Mô tả |
|--------|------|--------|
| `date` | string | Ngày giao dịch (yyyy-MM-dd) |
| `open` | number | Giá mở cửa (nghìn VND) |
| `high` | number | Giá cao nhất |
| `low` | number | Giá thấp nhất |
| `close` | number | Giá đóng cửa |
| `volume` | number | Khối lượng giao dịch |

<Note type="note">
Giá đã được chia 1000 so với dữ liệu thô từ API. Ví dụ: giá gốc 25500 → output 25.5 (nghìn VND).
</Note>
