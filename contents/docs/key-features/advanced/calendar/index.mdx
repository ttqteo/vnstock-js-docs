## Market Calendar

Module `market.calendar` cung cap thong tin ve ngay giao dich va gio mo/dong san HOSE.

### Kiem tra ngay giao dich

```typescript
import { market } from "vnstock-js";

market.calendar.isTradeDay("2026-04-03"); // true (Thu Sau)
market.calendar.isTradeDay("2026-04-04"); // false (Thu Bay)
market.calendar.isTradeDay("2026-01-01"); // false (Tet Duong lich)
```

### Ngay giao dich ke tiep / truoc do

```typescript
market.calendar.nextTradeDay("2026-04-03"); // "2026-04-06" (Thu Hai)
market.calendar.prevTradeDay("2026-04-06"); // "2026-04-03" (Thu Sau)
```

### Danh sach ngay nghi le

```typescript
market.calendar.holidays(2026);
// ["2026-01-01", "2026-02-15", "2026-02-16", ...]
```

### Gio giao dich san HOSE

```typescript
market.calendar.session();
// {
//   exchange: "HOSE",
//   morning: { open: "09:00", close: "11:30" },
//   afternoon: { open: "13:00", close: "14:45" },
//   ato: "09:15",
//   atc: "14:30"
// }
```

<Note>
Du lieu ngay nghi le duoc cap nhat thu cong theo Nghi dinh Chinh phu. Hien ho tro 2024-2026.
</Note>
